# Self-Supervised-Barlow-Twins-Pretraining-for-Data-Efficient-U-Net-Medical-Image-Segmentation
A two-phase pipeline that pretrains a U-Net encoder via Barlow Twins self-supervision on unlabelled images, then fine-tunes for segmentation using limited labelled data (5–50%). Benchmarked against supervised baselines using Dice/precision metrics across low-label regimes.
